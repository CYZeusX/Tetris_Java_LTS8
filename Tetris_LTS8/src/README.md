# Java 8 LTS Terminal Tetris & JVM Benchmark

A strict Java 8 implementation of Tetris showcasing Lambdas, Streams, `Optional`, and the `java.time` API. This project includes an interactive terminal mode and a `--headless` benchmark mode designed to stress test CPU and memory.

## Compiling the Project
Ensure you have a Java 8 JDK installed.
```bash
javac *.java
